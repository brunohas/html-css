TAGS HTML

Todas as tags HTML devem estar em < >, e fechar com </ > (existem algums que não precisa).

Tags não fechadas

  • <br> → quebra de linha
  • <hr> → linha horizontal

Tags de texto

  • <p> → parágrafo
  • <h1>, <h2>, ..., <h6> → títulos e sub-títulos
  • <wbr> → quebra de palavra
  • <style> → estilizar (CSS)
  • <strong> → negrito (exemplo)
  • <em> → itálico (exemplo)
  • <abbr> → abreviação (exemplo)
  • <del> → texto riscado (exemplo)
  • <ins> → texto sublinhado (exemplo)
  • <sup> → texto sobrescrito (x20+3)
  • <sub> → texto subscrito (H2O)
  • <adress> → informar endereço
  • <bdo> → inverte o texto
  • <bdi> → isola o conteúdo do conteúdo ao redor
  • <mark> → marca texto (exemplo)
  • <blockquote> → destaca citação indicando a fonte
  • <cite> → destaca citação
  • <q> → destaca citação curta/simples
  • <article> → "artigo" define um conteúdo
  • <aside> → indicar infomações sobre o conteúdo principal
  • <section> → "sessão" agrupar conteúdo
  • <header> → cabeçalho
  • <main> → conteúdo principal
  • <footer> → rodapé
  • <nav> → "menu de navegação"
  • <div> → define divisões lógicas
  • <pre> → definir que é um texto pré-formatado
  • <code> → indica que o texto é um código (monoespaçado)
  • <dfn> → marcação (exemplo)
  • <kbd> → elemento de entrada do teclado, entrada de voz ou qualquer dispositivo
  • <samp> → indentificar o texto que é uma amostra de saída de um programa
  • <var> → identificar uma váriavel em uma equação matemática ou programa
  • <select> → cria uma lista de opções
  • <option> → representa uma opção dentro de um elemento <select>

Tags para links

  • <a> → cria hiperligação em páginas web, arquivos, endereço de e-mail, entre outros
  • <base> → especificar a URL base ou o local de destino para todas as URLs relativas dentro do documento

Tags para imagens, áudio, vídeo e objeto

  • <img> → inserir imagens em uma página
  • <picture> → para fazer imagem dinâmica
  • <map> → define uma área clicável no mapa de imagem (usado com <area>)
  • <area> → define uma área clicável dentro da imagem (usado dentro do <map>)
  • <object> → define um contêiner para um recurso externo, como uma imagem, página web, reprodutor de mídia ou app plug-in
  • <audio> → inserir áudio em uma página
  • <video> → inserir vídeo em uma página
  • <iframe> → incorporar um outro documento HTML dentro de uma página

Tags para listas

  • <ul> → lista não ordenada
  • <ol> → lista ordenada
  • <li> → representar um item da lista (dentro da tag <ul> ou <ol>)
  • <dl> → representa uma lista de associação, lista de descrição
  • <dt> → define um termo ou nome em uma lista de descrições
  • <dd> → define a descrição de um termo em uma lista de descrição

Tags para tabelas simples

  • <table> → abre a tabela
  • <caption> → título/legenda da tabela
  • <tr> → linha da tabela
  • <td> → dado da tabela
  • <th> → dado do cabeçalho da tabela

Tags para tabelas grandes/complexas

  • <table> → abre a tabela
  • <caption> → título/legenda da tabela
  • <thead> → cabeçalho da tabela → tr → td ou th
  • <tbody> → corpo da tabela → tr → td ou th
  • <tfoot> → rodapé da tabela → tr → td ou th

Tags para formulários

  • <form> → abre o formulário
  • <fieldset> → agrupar o formulário
  • <legend> → título/legenda ao grupo
  • <input> → pergunta
  • <label> → faz a ligação com o <input>
  • <output> → mostra na tela

TAGS CSS

Todas as tags de estilo devem estar dentro da tag <style> (ou dentro de um arquivo criado em CSS) ou em uma tag específica que deseja estilizar.

CSS para ajustes de texto

  1. content: → mostra um texto pré-definido
  2. text-align: → alinhamento horizontal do texto (left / right / center / justify)
  3. vertical-align: → alinhamento vertical do texto (text-top / sub / bottom / middle / top / baseline)
  4. text-decoration: → decoração adicional (underline / overline / line-through)
  5. text-indent: → espaço na primeira linha do parágrafo (20px ~ 40px)
  6. text-transform: → controle de maiúsculas e minúsculas (uppercase / lowercase / capitalize)
  7. text-shadow: → sombra no texto (0px 0px 0px cor)
  8. line-heigth: → espaço entre as linhas de um parágrafo
  9. color: → cor do texto
  10. font-style: → estilo da fonte (normal / italic / oblique)
  11. font-weight: → "peso" da fonte (lighter / normal / bold / bolder)
  12. font-size: → tamanho da fonte
  13. font-family → tipo de fonte
  14. font-variant: → especificar que o texto será exibido em fonte versalete (normal / small-caps = exemplo de fonte versalete)
  15. font: → abreviação para {font-style: → font-weigth: → font-size: → font-family:}

CSS para ajuste de fundo

  1. background-color: → cor de fundo
  2. background-image: linear-gradient( ) → degradê em linha (to top / to right / to bottom / to left + cores)
  3. background-image: radial-gradient( ) → degradê em circulo (circle + cores)
  4. background-image: url( ) → imagem de fundo
  5. background-position: → posição da imagem - 1° vertical 2° horizontal (center / top / rigth / bottom / left)
  6. background-repeat: → se repete ou não a imagem (no-repeat / repeat / repeat-x / repeat-y / space / round)
  7. background-size: → tamanho da imagem
  8. background-attachment: → se fixa ou rola a imagem (fixed / scroll)
  9. background-clip: → define a área da box onde a imagem de fundo é aplicada (padding-box / border-box / content-box)
  10. background-origin: → define a posição de origem da imagem (padding-box / border-box / content-box )
  11. background: → abreviação para {background-color: → background-image: → background-position: → background-repeat: → [background-size] → background-attachment:}

CSS para ajuste de borda

  1. border-width: → define a espessura da borda
  2. border-style: → define o estilo da borda (none / solid / dotted / dashed / double / groove / ridge / inset / outset)
  3. border-color: → define a cor da borda
  4. border: → abreviação para {border-width: → border-style: → border-color:}
  5. border-radius: → arredonda as pontas da borda
  6. border-image-source: url( ) → borda decorada
  7. border-image-slice: → fatiar a imagem (+/- 20)
  8. border-image-repeat: → repetir a imagem (repeat / stretch)
  9. border-image: → abreviação para {border-image-source: → border-image-slice: → border-image-repeat:} precisar ter uma borda
  10. border-image-width: → defini a espessura das bordas
  11. border-image-outset: → define uma área externa á área da borda para onde a imagem será deslocada
  12. border-collapse: → junta as borda (collapse / separate)
  13. box-shadow: → sombra de uma box (0px 0px 0px cor)
  14. box-sizing: border-box → permite incluir o preenchimento e a borda na largura e altura de um elemento

  2. list-style-position: → define a posição do marcador de lista (outside / inside)

display:

O atributo display: define o comportamento do elemento em relação aos demais elementos do documento.

  1. display: flex; → permite organizar elementos em uma página da web de forma flexível
  2. display: none; → faz com que o elemento HTML não seja renderizado (apaga na visualização)
  3. display: block; → faz com que o elemento HTML seja renderizado como bloco (ocupa linha inteira)
  4. display: inline; → faz com que o elemento HTML seja renderizado inline, dentro do bloco na mesma linha (não ocupa linha inteira)
  5. display: inline-block; → funciona somente para os elementos HTML que sejam display: inline;
  6. display: list-item; → faz com que o elemento HTML seja renderizado como um item de lista (irá aparecer um marcador de lista)
  7. display: run-in; → exibe um elemento como block ou inline, dependendo do contexto
  8. display: table; → faz com que o elemento HTML seja renderizado como tabela

position:

A propriedade position: define como um elemento deve ser posicionado na página. Ela funciona em conjunto com as propriedades top, bottom, left, right para determinar a posição final de um elemento.

  1. position: static; → posiciona o elemento no canto superior esquerdo do documento (padrão)
  2. position: fixed; → o elemento fica fixo na tela, mesmo que a página tenha rolagem (pode ser usado para criar menus de navegação que continuam visíveis)
  3. position: sticky; → o elemento fica fixo em relação ao rolamento da página, mas não em relação à tela
  4. position: relative; → o elemento não muda de posição, mas é possível alterar a sua posição em relação à sua posição inicial
  5. position: absolute; → o elemento pode ter 2 comportamentos diferentes, dependendo do valor do elemento pai

A propriedade position: pode ser acompanhada de outras, tais como, top, right, bottom, e left. Essas propriedades determinam como ficará a localização final do objeto. O valor postition: static; ignora o valor dessas propriedades.

transform:

A propriedade transform: permite modificar o espaço coordenado do modelo de formatação CSS. Usando-a, elementos podem ser traduzidos, rotacionados, ter seu tamanho ajustado e inclinados de acordo com os valores definidos.

  1. transform: matrix( ) → especifica uma matriz de transformação 2D composta dos 6 valores especificados (n°, n°, n°, n°, n°, n°).
  2. transform: rotate( ) → gira o elemento no sentido horário em torno de sua origem pelo ângulo especificado (n°deg).
  3. transform: scale( ) → especifica uma operação de escala 2D descrita por [sx, sy]. Se sy não for especificado, vai ser igual ao sx (n°, n°).
  4. transform: scaleX( ) → especifica uma operação de escala usando o vetor [sx, 1] (n°).
  5. transform: scaleY( ) → especidica uma operação de escala usando o vetor [1, sy] (n°).
  6. transform: skew( ) → inclina o elemento em torno dos eixos X e Y pelos ângulos especificados (n°deg, n°deg).
  7. transform: skewX( ) → inclina o elemento em torno do eixo X pelo ângulo escolhido (n°deg).
  8. transform: skewY( ) → inclina o elemento em torno do exio Y pelo ângulo escolhido (n°deg).
  9. transform: translate( ) → especifica uma translação 2D pelo vetor [tx, ty]. Se ty não for especificado, vai ser igual ao tx (n°, n°)
  10. transform: translateX( ) → traduz o elemento pela quantidade fornecida ao longo do eixo X (n°)
  11. transform: translateY( ) → traduz o elemento pela quantidade fornecida ao longo do eixo Y (n°)
  12. transform-origin: posição + posição + n°; → rotação em 3D

Se a propriedade tem um valor diferente de none, um contexto de empilhamento será criado. Neste caso, o objeto atuará como um bloco recipiente para position: fixed; para os elementos que estão contidos.

transition:

A propriedade transition: é um mecanismos que permitam a mudança de estilização de um elemento HTML de uma forma suave e controlada no tempo.

  1. transition-property: → define as propriedades CSS às quais serão aplicadas as transições (ex: div {transition-property: border, color;})
  2. transition-duration: → define o tempo que uma animação de transição deve levar para ser concluída (ex: div{transition-duration: 0.5s})
  3. transition-timing-function: → define a forma como a transição progride no tempo (ex: div{transition-timing-function: linear;})
  4. transition-delay: → define o tempo de espera em segundos para início da transição (div{transition-delay: 2s;})
  5. transition: → abreviação para {transition-property: → transition-duration: → transition-timing-function: → transition-delay}

translate:

A propriedade translate: permite que você especifique transformações translate: individualmente e independentemente da propriedade transform:. É mais adequado para o uso de interface de usuário, e nos poupa quando se trata de lembrar a ordem exata que devemos especificar nos valores das funções de transform:.

  1. translate: none; → especifica que nenhum deslocamento deva ser aplicado
  2. translate: n°; → (único valor) especifica o deslocamento ao longo do eixo X
  3. translate: n°, n°; → (2 valores) especifica o deslocamento ao longo dos eixos X e Y (respectivamente) de uma 2D translation
  4. translate: n°, n°, n°; → (3 valores) especifica o deslocamento dos eixos X, Y e Z (respectivamente) de uma 3D translation

calc( )

A função calc( ) permite que se definam valores CSS com uso de expressões matemáticas, ou seja, o valor adotado para a propriedade é o resultado de uma expressão matemática.

Os operadores matemáticos válidos são: + (soma), - (subtração), * (multiplicação) e / (divisão). As unidades de medida CSS válidas na expressão matemática são as unidades CSS para: comprimento, ângulo, tempo, frequência e números inteiros e fracionários. Na expressão matemática que define o valor CSS é permitido que se misture diferentes unidades de medida. A sintaxe geral para uso desta função é conforme mostrada a seguir: 


                                seletor { 
                                    propriedade: calc(expressão matemática);
                                }
