Exemplos de Flexbox

flex-direction - (direção em que os elementos flexíveis serão exibidos)

flex-direction: row; (alinha da esquerda p/ direita) - padrão

            
                <body>                                         |        <style>
                    <div class="container">                    |            div.container {
                        <div class="item">A</div>              |                background-color: lightsalmon;
                        <div class="item">B</div>              |                display: flex;
                        <div class="item">C</div>              |                flex-direction: row;
                        <div class="item">D</div>              |            }
                        <div class="item">E</div>              |            div.item {
                        <div class="item">F</div>              |                background-color: lightskyblue;
                        </div>                                 |                border: 1px solid black;
                    </body>                                    |                width: 200px;
                                                               |            }
                                                               |        </style>

flex-wrap - (se os itens flexíveis quebram de linha ou não)

flex-wrap: nowrap; (não permite quebra de linha) - padrão

Obs: Esse exemplo esta com flex-direction: row;. 

            
                <body>                                         |        <style>
                    <div class="container">                    |            div.container {
                        <div class="item">A</div>              |                background-color: lightsalmon;
                        <div class="item">B</div>              |                display: flex;
