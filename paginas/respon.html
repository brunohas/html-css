RESPONSIVIDADE

A responsividade de um site é a capacidade de se adaptar a diferentes tamanhos de tela. O objetivo é proporcionar uma boa experiência de visualização ao usuário, independentemente do dispositivo que esteja sendo utilizado. Para saber melhor a medida ou visualizar em tela de celular, utilizar a extensão do Google Chrome Window Resizer. Clique aqui para ir no meu primeiro projeto com responsividade.

Responsividade da tela

Para criar essa responsividade, utilizamos os parâmetros min-width: (largura mínima da tela) e max-width: (largura máxima da tela). Normalmente a largura mínima será de 320px (pois é a largura de alguns celulares mais antigos). 


                                            <style>
                                                * {
                                                    margin: 0px;
                                                    padding: 0px;
                                                }
                                                
                                                body, html {
                                                    heigth: 100vh;
                                                    min-width: none;
                                                    max-width: 100vw;
                                                }
                                            </style>

Responsividade da imagem

Para fazer a responsividade de uma imagem, adicionar a tag de estilo somente para a imagem (img {width: 100%;}) para que ela "quebre" a largura junto com a janela. 


                                            <style>
                                                img {
                                                    width: 100%;
                                                    display: block;
                                                    margin: auto;
                                                    max-width: largura da imagem;
                                                }

Ou também podemos utilizar a tecnica da imagem dinâmica mostrada na página Imagens.

Responsividade de vídeo

Para fazer a responsividade de um vídeo, envelopamos a tag de <video> ou <iframe> com uma tag <div>, e adicionamos um class=" " para identificar que é um vídeo. Depois criamos 2 parâmetros na tag de estilo, div. {} e div. > video {} ou div. > iframe {}, para colocar as seguintes tags: 


        <div class="video">                                  |    <div class="video">
            <video src=" "></video>                         ou        <iframe src=" " frameborder="0"></iframe>
        </div>                                               |    </div>

                    <style>
                        div.video {
                            padding-bottom: %; (50%)          !- parte preta dentro do vídeo -!
                            position: relative;               !- posição da tag <div> -!
                        }

                        div.video > video {
                            position: absolute;               !- posição do video dentro da tag <div>
                            top: %; (5%)                      !- distância do topo -!

                    Colocar as medidas em porcentagem (%), para que consiga fazer a responsividade.

Typical Device Breakpoints

Os Breakpoints de dispositivo são pontos de interrupção que determinam como um site se adapta a diferentes tamanhos de tela. Alguns pontos de interrupção comuns para dispositivos em 2025 incluem:

  1. Dispositivos extra pequenos → telefones, 600px e inferiores (@media screen and (max-width: 600px))
  2. Dispositivos pequenos → tablets retratos e telefones grandes, 600px e superiores (@media screen and (min-width: 600px))
  3. Dispositivos médios → tablets em paisagem, 768px e superiores (@media screen and (min-width: 768px))
  4. Dispositivos grandes ↠ laptops/desktops, 992px e superiores (@media screen and (min-width: 992px))
  5. Dispositivos extragrandes → laptops e desktops grandes, 1200px e superiores (@media screen and (min-width: 1200px))

MEDIA QUERY

Media Query é um módulo do CSS que permitem ajustar a visualização de uma página para diferentes dispositivos. Elas são utilizadas para tornar um site responsivo, adaptando a apresentação do conteúdo a uma gama específica de dispositivos. As media queries permitem definir expressões que limitam o escopo das folhas de estilo, aplicam especificidades para celulares, tablets e desktops, determinam a visualização das informações contidas na página, detecta carecterísticas importantes do dispositivo, como largura e altura da viewport, orientação da tela e resolução da mesma.

Resumindo, media queries são vários links de CSS do mesmo programa, porém para definições difirentes tipo, um CSS para demonstração da tela pequena, outro CSS para demonstração de tela grande, e outro CSS para impressão. Adicionamos um parâmetro media=" " (media="all" → adequa o layout para todos os tipos de mídia / media="screen" → define como o layout será exibido em diferentes dispositivos, como desktops, laptops, tablets e smartphones / media="print" → exclusivo para impressoras) dentro da tag <link:css>. 


        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Media Query</title>
            <link rel="stylesheet" href="tudo.css" media="all"          !- link:css para todos os tipos de mídia -!
            <link rel="stylesheet" href="tela.css" media="screen"       !- link:css para as telas -!
            <link rel="stylesheet" href="impressora.css" media="print"  !- link:css para impressão -!
        </head>
        <body>
    
        </body>

MEDIA FEATURES

As media features são usadas em conjunto com as media queries, elas são úteis para criar sites responsivos, que se adaptam a diferentes tamanhos de tela, orientações do dispositivo e preferências do usuário, são adicionadas entre (parênteses) junto com o parâmetro media=" ", por exemplo: 


    <!DOCTYPE html>
    <html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Media Features</title>
        <link rel="stylesheet" href="tudo.css" media="all"                                       !- CSS principal -!
        <link rel="stylesheet" href="retrato.css" media="screen and (orientation: portrait)"     !- CSS p/ aparelho deitado -!
        <link rel="stylesheet" href="impressora.css" media="screen and (orientation: landscape)" !- CSS p/ aparelho em pé -!
    </head>
    <body>

    </body>

Elas podem ser usadas para identificar características do dispositivo, identificar preferências do usuário, adaptar sites e aplicações de acordo com as preferências do usuário, definir regras para dispositivos em retrato ou paisagem. Agora segue alguns tipos de media features:

  • (width) → largura do viewport
  • (heigth) → altura do viewport
  • (orientation) → orientação do viewport
  • (aspect-radio) → proporção da largura para a altura do viewport
  • (resolution) → densidade de pixels do dispositivo
  • (prefers-color-scheme) detecta se o tema é claro ou escuro
  • (prefers-reduced-motion) detecta se o usuário prefere animações reduzidas
  • (inverted-colors) → detecta se o navegador tem inversão de cores

FLEXBOX

O Flexbox é uma funcionalidade do CSS que permite organizar elementos em uma página da web de forma flexível, podemos organizar elementos em linhas e colunas, alinhar elementos horizontalmente e verticalmente, criar layouts dinâmicos, criar páginas que funcionam bem em diferentes tamanhos de tela, etc. Para usar o flexbox, é necessário especificar um container (pai) e atribuir a propriedade display: flex;, e para os itens (filho) que estão dentro do container, atribuir a propriedade flex: auto; (não permite espaço em branco dentro do elemento pai) para que consiga ficar flexível. Por exemplo: 


            <body>                                     |        <style>
        (pai) → <div class="container">                |            div.container {
        (filho)  →  <div class="item">A</div>          |                background-color: lightsalmon;
        (filho)  →  <div class="item">B</div>          |                display: flex;
        (filho)  →  <div class="item">C</div>          |            }
        (filho)  →  <div class="item">D</div>          |            div.item {
        (filho)  →  <div class="item">E</div>          |                border: 1px solid black;
        (filho)  →  <div class="item">F</div>          |                text-align: center;

flex-direction

A propriedade flex-direction do CSS define a direção em que os elementos flexíveis serão exibidos. Ela é usada para definir o eixo principal e a direção (normal ou invertida).

  1. flex-direction: row; → alinha os elementos em linha, da esquerda para direita (padrão).
  2. flex-direction: row-reverse; → alinha os elementos em linha, da direita para esquerda.
  3. flex-direction: column; → alinha os elementos em uma única coluna, de cima para baixo.
  4. flex-direction: column-reverse; → alinha os elementos em uma única coluna, de baixo para cima.

flex-wrap

A propriedade CSS flex-wrap define se os itens flexíveis devem ser forçados a ficar em uma única linha ou se podem ser quebrados em várias linhas. Para usar o flex-wrap, é preciso considerar o tamanho dos itens e do container, para que a distribuição dos elementos seja adequada ao design do layout.

  1. flex-wrap: nowrap; → não permite a quebra de linha (padrão).
  2. flex-wrap: wrap; → quebra de linha quando um dos itens flexíveis não puder mais ser compactado.
  3. flex-wrap: wrap-reverse; → quebra de linha na direção contrária, ou seja, para linha de acima.

flex-flow

A propriedade CSS flex-flow é uma propriedade shorthand (uma mesma declaração inclui vários valores relacionados a mais de uma propriedade) que inclui flex-direction e flex-wrap. Segue todos os tipos de flex-flow:

  1. flex-flow: row nowrap; → alinha da esquerda para direita + não permite quebra de linha (padrão).
  2. flex-flow: row wrap; → alinha da esquerda para direita + quebra de linha quando item flexível não puder ser mais compactado.
  3. flex-flow: row wrap-reverse; → alinha da esquerda para direita + quebra de linha ao contrário, para linha de cima.
  4. flex-flow: row-reverse nowrap; → alinha da direita para esquerda + não permite quebra de linha.
  5. flex-flow: row-reverse wrap; → alinha da direita para esquerda + quebra de linha quando item flexível não puder ser mais compactado.
  6. flex-flow: row-reverse wrap-reverse; → alinha da direita para esquerda + quebra de linha ao contrário, para linha de cima.
  7. flex-flow: columns nowrap; → alinha em uma única coluna de cima para baixo + não permite quebra de linha.
  8. flex-flow: columns wrap; → alinha em uma única coluna de cima para baixo + quebra de linha quando item flexível não puder ser mais compactado.
  9. flex-flow: columns wrap-reverse; → alinha em uma única coluna de cima para baixo +
  10. flex-flow: columns-reverse nowrap; → alinha em uma única coluna de baixo para cima + não permite quebra de linha.
  11. flex-flow: columns-reverse wrap; → alinha em uma única coluna de baixo para cima + quebra de linha quando item flexível não puder ser mais compactado.
  12. flex-flow: columns-reverse wrap-reverse; → alinha em uma única coluna de baixo para cima + quebra de linha ao contrário, para linha de cima.

Obs: Para fazer quebra de linhas em colunas, adicionar um valor de height no elemento pai.

justify-content

A propriedade CSS justify-content faz o alinhamento em relação ao eixo principal (main-axis), se atentar a direção do eixo (flex-direction).

  1. justify-content: flex-start; → alinha os elementos no início do eixo principal, grudado no main-start (padrão).
  2. justify-content: flex-end; → alinha os elementos no final do eixo principal, grudado no main-end.
  3. justify-content: center; → alinha os elementos no meio do eixo principal.
  4. justify-content: space-between; → distribui os elementos uniformemente, com espaço entre eles.
  5. justify-content: space-around; → distribui os elementos uniformemente, com espaço igual ao redor deles.
  6. justify-content: space-evenly; → distribui os elementos de modo que o espaçamento entre quaisquer dois seja igual.

align-items

A propriedade CSS align-items faz o alinhamento em relação ao eixo transversal (cross-axis), se atentar a direção do eixo (flex direction).

  1. align-items: stretch; → estica o elemento para que caiba no container (padrão).
  2. align-items: flex-start; → alinha os elementos no início do eixo transversal, grudado no cross-start.
  3. align-items: flex-end; → alinha os elementos no final do eixo transversal, grudado no cross-end.
  4. align-items: center; → alinha os elementos no centro do eixo transversal.

align-content

A propriedade CSS align-content serve para organizar as linhas de um contêiner flexível quando há espaço extra no eixo transversal, só pode ser utilizado quando tiver quebra de linha (flex-wrap: wrap; ou flex-wrap: wrap-reverse;).

  1. align-content: stretch; → distribui as linhas uniformemente ao longo do eixo transversal (padrão).
  2. align-content: flex-start; → distribui as linhas a partir do início do eixo transversal (cross-start).
  3. align-content: flex-end; → distribui as linhas a partit do im do eixo transversal (cross-end).
  4. align-content: center; → mantém as linhas no centro do eixo transversal.
  5. align-content: space-between; → distribui as linhas uniformemente entre o início e o fim do eixo transversal.
  6. align-content: space-around; → distribui as linhas uniformemente ao longo do eixo transversal, atribuindo margens iguais dos lados.
  7. align-content: space-evenly; → distribui as linhas igualmente com o mesmo espaçamento entre elas.
