TIPOS DE FONTE

Nem todo computador tem todos os tipos de fonte, por isso, na hora de escolher a fonte para o site, colocar mais de um tipo da mesma fonte e também colocar a família da fonte por último, pois se o dispositivo não achar essa fonte, ela pula para a outra. Os 3 tipos de família de fonte mais utilizadas são: sans-serif (sem serifa), serif (serifada) e monospace (monoespaçada → letras e caracteres ocupam o mesmo espaço horizontal).
Ex:

Fonte sem serifa (sans-serif)

  • Arial → AaBbCcDd
  • Verdana → AaBbCcDd
  • Trebuchet MS → AaBbCcDd
  • Segoe UI → AaBbCcDd

Fonte serifada (serif)

  • Times New Roman → AaBbCcDd
  • Georgia → AaBbCcDd
  • Cambria → AaBbCcDd
  • Cochin → AaBbCcDd

Fonte Monoespaçada (monospace)

  • Corrier New → AaBbCcDd

  • monospace → AaBbCcDd

Outros tipos de fontes que não são muito utilizadas são as fontes cursivas (simulam manuscrito) e fantasia ou fantasy (fontes decorativas). Porém não tem muitas opções delas, mas conseguimos importar fontes do site Google Fonts ou DaFont (também tem fontes não serifadas, serifadas e monoespaçadas), basta selecionar a fonte desejada e copiar o @import clicando na opção Get font → Get embed code, depois colar na tag de estilo.
Ex:

Fonte cursiva (cursive)

  • cursive → AaBbCcDd

  • Fonte importada do Google Fonts → AaBbCcDd

Fonte fantasia (fantasy)

  • fantasy → AaBbCcDd
  • Fonte importada do Google Fonts → AaBbCcDd

Existem 4 tipos de alinhamentos, utilizamos o parâmetro text-aling: (left = esquerda / right = direita / center = centro). Também conseguimos utilizar o text-aling: justify; para fazer o alinhamento nos 2 lados do texto. E para fazer aquele espaço no começo do parágrafo, utilizar o parâmetro text-indent: 20px;, e para fazer um espaço entre as linhas do parágrafo, utilizar o parâmetro line-height:. Não esquecer que todos esses parâmetros de CSS tem que estar dentro da tag <style>.

FONTES EXTERNAS

Nós conseguimos importar e baixar fontes externas, importando ou baixando elas do Google Fonts ou outro site. Para importar uma fonte, já foi explicado no texto acima, basta selecionar a fonte desejada e copiar o @import clicando na opção Get font → Get embed code, depois colar na tag de estilo.
Ex: 

                    
    <!DOCTYPE html>
    <html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Fontes do Google</title>
        <style>
            @import url('https://fonts.googleapis.com/css2?family=Playwrite+AU+SA:wght@100..400&family=Rubik+Vinyl&
            display=swap');
            @import url('https://fonts.googleapis.com/css2?family=Comforter+Brush&display=swap');
        </style>
    </head>
    <body>
        <p style="font-family: 'Rubik Vinyl'; text-align: center;">
            Segue o exemplo de importação de uma fonte do Google Fonts.
        </p>
        <p style="font-family: Comforter Brush; text-align: center;">
            Segue o exemplo de importação de outra fonte do Google Fonts.
        </p>
    </body>

Já para baixar um fonte, baixamos o arquivo.zip e extraímos na mesma pasta do código-fonte, colocamos @font-face dentro da tag <style>, irá abrir 2 parâmetros: font-family: → nome para a fonte (colocar entre '')/ src: url(); → colocar o nome igual a do arquivo extraído, depois acrescentar outro parâmetro format() → opentype (otf) / truetype (ttf) / embedded-opentype / truetype-aat (Apple Advanced Typography) / svg, colocar de acordo com o arquivo extraído (ver nas propriedades do arquivo).
Ex: 

                    
    <!DOCTYPE html>
    <html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Fontes Externas</title>
        <style>
            @font-face {font-family: 'OldLondon';
            src: url('fonts/OldLondon.ttf') format('truetype'), url('fonts/OldLondonAlternate.ttf') format('truetype'), 
            url('fonts/Olondon_.otf') format('opentype'), url('fonts/Olondona.otf') format('opentype');}
        </style>
    </head>
    <body>
        <p style="font-family: 'OldLondon'; text-align: center;">
            Segue o exemplo de uma importação de uma fonte baixada.
        </p>
    </body>

DETECTANDO UMA FONTE

Para conseguir localizar uma fonte de um site, utilizar o Fonts Ninja (extensão do Google Chrome), basta entrar no site em que você deseja saber qual a fonte, e clicar no ícone do Fonts Ninja. Enquanto a extensão estiver aberta, basta passar o mouse encima do texto para saber qual sua fonte, tamanho, cor, etc...

Para conseguir localizar uma fonte de uma imagem, utilizamos os sites WhatFontIs ou FontSquirrel ou MyFonts. Basicamente todos funcionam arrastando a imagem até a área do site, e depois fazer algumas edições para a imagem ficar melhor de se visualizar.

Modelos de Caixas

É aplicavel ao HTML e CSS, é um conjunto para facilitar a organização do site. Básicamente cada caixa é o conteúdo que está em uma tag, por exemplo, <h1>Olá, Mundo!</h1> → Olá, Mundo! (é a caixa). A caixa tem um tamanho e é especificado em 2 medidas muito importantes: height: (altura) e width: (largura). Também existem outras medidas: border (borda), padding(acolchoamento / preenchimento), outline(contorno / traçado), margin(margem). Segue a anatomia de uma caixa:

Para entender melhor, na página do navegador pressionar Ctrl + Shift + i ou clicar com o mouse direito na página e clicar em "Inspecionar" para abrir o DevTools (se editar no DevTools, não edita no código original).

Tipos de Caixas

  • box-level → é uma caixa que quebra de linha, ocupa a largura inteira (width), e quebra de linha para dar a continuidade do conteúdo.
    Exemplos de tags: <div> / <h1>~<h6> / <p> / <main> / <header> / <nav> / <video> / ...
  • inline-level → é uma caixa que não quebra de linha, não ocupa a largura inteira (width), e da a continuidade do conteúdo na mesma linha.
    Exemplos de tags: <span> / <a> / <code> / <strong> / <em> / <sub> / <sup> / ...

Também conseguimos mudar um box-level para inline-level e vice-versa, basta acrescentar uma tag de estilo display: block(box-level) ou display: inline (inline-level).

ID e CLASS

ID

O ID serve para identificar um parâmetro, por exemplo, em um código existem mais de um <h1>, o primeiro <h1> é o título do site, e os outros são os títulos do texto, para identificar que o primeiro <h1> não vai ter as mesmas características, abrimos um parâmetro (id=" ") dentro da tag <h1>. E depois na tag de estilo abrimos (h1#nome do id) para conseguir estilizar somente a tag <h1 id=" ">. (OBS: em HTML é id=" " em CSS é #). 

 
        <body>                                                |    <style>
            <h1 id="titulo">Título principal</h1>             |        h1#titulo {
            <h1>Título texto 1</h1>                           |            font-size: 2em;    !- somente o h1 com o id -!
                <p>Texto 1</p>                                |            font-weigth: bold;
            <h1>Título texto 2</h1>                           |        }
                <p>Texto 2</p>                                |        h1 {
            <h1>Título texto 3</h1>                           |            font-size:1.5em;   !- todos os h1 que não tem o id -!
                <p>Texto 3</p>                                |            font-weigth: normal;
        </body>                                               |        }

CLASS

O id só podemos utilizar 1 vez por tag (podemos utilizar vários id com nomes diferentes), já o class, conseguimos utilizar para mais de uma tag. Utilizamos igual o id, colocar o class=" " dentro da tag desejada, e dar um nome a ele. Já na tag de estilo abrimos com um ponto (.nome do class) para conseguir estilizar as tags que estejam com class=" ". Agora olhe um exemplo, quero cor de fundo somente para o sub-título do texto 2 e o sub-título do texto 3. (OBS: em HTML é class=" " em CSS é . ). 


    <body>                                                |    <style>
        <h1>Título principal</h1>                         |        h2 {                !- somente o h2 sem class (do texto 1) -!
        <h2>sub-título texto 1</h2>                       |            color: black;
            <p>texto 1</p>                                |        }
        <h2 class="destaque">sub-título texto 2</h2>      |        h2.destaque {       !- os h2 que tem o class="destaque" -!
            <p>texto 2</p>                                |            background-color: darkgray;
        <h2 class="destaque">sub-título texto 3</h2>      |            color: white;
            <p>texto 3</p>                                |        }
    </body>                                               |    </style>

PSEUDO-CLASSES

Enquanto em HTML usa id=" " CSS utiliza (#), e em HTML usa class=" " CSS utiliza ( . ), as pseudo-classes utilizam ( : ) no CSS. As pseudo-classes são usadas para definir um estado especial de um elemento. Utilizar no elemento em que você deseja, alguns tipos de pseudo-classes:

Pseudo-classes comumente usadas com links

  1. :hover → define o estilo de exibição do elemento quando passa o mouse por cima
  2. :visited → define o estilo de links já visitados
  3. :active → define o estilo de exibição do elemento quando ativado (quando clica)
  4. :link → define o estilo do link que não foi visitado

Pseudo-classes comumente usadas com formulários

  1. :valid → seleciona qualquer input válido (conteúdo apropriado)
  2. :invalid → selecionar todos inputs inválidos (conteúdo no formato incorreto)
  3. :checked → define o estilo de campos do tipo radio, checkbox ou option quando selecionado
  4. :focus → define o estilo do campo quando estiver em foco (campo em edição / selecionado)
  5. :default → define o estilo de campos de formulário entre um grupo de elementos similares
  6. :disabled → define o estilo de campos de formulário que tenham sido desabilitados
  7. :empty → define o estilo dos elementos que não tem qualquer tipo de conteúdo
  8. :enabled → define o estilo dos elementos que estão habilitados (selecionado / clicado / digitado / etc)
  9. :in-range → define o estilo dos elementos que definem uma faixa de valores e cujos valores estejam dentro da faixa especificada
  10. :out-of-range → define o estilo dos elementos que definem uma faixa de valores e cujos valores estejam fora da faixa especificada
  11. :indeterminate → define o estilo dos elementos input do tipo radio buttons e check boxes que foram desmarcados pelo usuário ou que foram carregados na página sem estarem marcados
  12. :optional → define o estilo de campos do tipo input que não são de preenchimento obrigatório, ou seja, campos que não contêm o atributo required
  13. :read-only → define o estilo dos elementos que não podem ser editados pelo usuário. É idêntica à pseudoclasse :disabled e o atributo que foi definido na marcação HTML é quem determina qual pseudoclasse usar.
  14. :read-write → define o estilo dos elementos que podem ser editados pelo usuário. Casa também com elementos aos quais foi definido o atributo contenteditable na marcação HTML
  15. :required → define o estilo dos elementos input aos quais foi definido o atributo required na marcação HTML.

Pseudo-classes comumente usadas em listas e tabelas

  1. :nth-child(even) → usado para fazer efeito zebrado (2n)
  2. :nth-child(odd) → usado para fazer efeito zebrado (2n-1)

Pseudo-classes estruturais

  1. :first-child → corresponde ao primeiro elemento filho de seu elemento pai
  2. :last-child → corresponde ao último elemento filho de seu elemento pai
  3. :first-of-type → corresponde ao primeiro elemento filho de um determinado tipo que esteja em seu elemeto pai
  4. :last-of-type → corresponde ao último elemento filho de um determinado tipo que esteja em seu elemento pai
  5. :root → corresponde ao elemento raiz do documento
  6. :not → pseudoclasse de negação
  7. :target → corresponde ao elemento ao qual foi definido um atributo ID
  8. :only-child → corresponde ao filho único de um elemento.
  9. :only-of-type → casa com elementos que não têm irmãos do seu tipo. Ele é idêntico ao elemento :only-child exceto pelo fato de que casa com um determinado tipo de elemento o que faz com que o seletor seja mais específico.
  10. :nth-child( ) → casa com um ou mais elementos dependendo da ordem em que eles aparecem na marcação HTML
  11. :nth-last-child( ) → é idêntica à pseudoclasse :nth-child exceto que o casamento se dá contando-se os elementos filhos de trás para frente.
  12. :nth-of-type( ) → é idêntica à pseudoclasse :nth-child, a principal diferença é que se trata de um seletor mais específico, pois casa com elementos filhos de determinado tipo
  13. :nth-last-of-type( ) → é idêntica à pseudoclasse :nth-of-type exceto que o casamento se dá contando-se os elementos filhos de trás para frente.

PSEUDO-ELEMENTOS

O pseudo-elemento pode agir nas classes, nos elementos e nas ids, mexem no conteúdo periférico do elemento. Para pseudo-elemento utilizar ( :: ), conseguimos colocar conteúdo sem o HTML, normalmente utiliza ( ::before / ::after ) para saber se é para colocar o conteúdo antes ou depois.

::before

O pseudo-elemento ::before adiciona conteúdo antes do conteúdo do elemento selecionado. 


                    main::before {
                        content: 'comentário';
                    }

O content: 'comentário' irá aparecer antes do conteúdo <main>.

::after

O pseudo-elemento ::after adiciona conteúdo depois do conteúdo do elemento selecioando. 


                    main::after {
                        content: 'comentário';
                    }

O content: 'comentário' irá aparecer depois do conteúdo <main>.

::first-letter

O pseudo-elemento ::first-letter personaliza somente a primeira letra de um elemento. 


                <p>                                 |        p::first-letter {
                    um exemplo de first-letter      |            font-size: 1.5em;
                </p>                                |            font-style: italic;
                                                    |        }

Somente a letra "u" irá receber o estilo.

::first-line

O pseudo-elemento ::first-line personaliza o conteúdo da primeira linha de um elemento. 


            <p>                                      |        p::first-line {
                um exemplo de first-line             |            text-decoration: underline;
                que estiliza somente primeira linha  |            font-style: italic;
            </p>                                     |        }

Somente a primeira linha "um exemplo de first-line" irá receber o estilo.

Símbolos que podemos usar na CSS que vimos:

  • # → id
  • . → class
  • : → pseudo-class
  • :: → pseudo-element
  • > → children (filho)
