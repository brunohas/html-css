FORMULÁRIOS

Um formulário é um local onde o visitante consegue colocar dados. Para criar um formulário, utilizamos a tag <form>, todos os dados tem que estar entre a tag <form> e </form>, para perguntar os dados do visitante, acrescentamos a tag <input> e para fazer uma ligação com o input acrescentamos a tag <label>. Também conseguimos adicionar os parâmetros required para que o campo não possa ser enviado sem nenhum caractere (conseguimos mudar o mínimo e o máximo de caracteres, minlength=" " e maxlength=" "), placeholder=" " para detalhar a caixa de pergunta, size para mudar o tamanho da caixa, autocomplete=" " para completar com os dados salvos pelo navegador, e também conseguimos agrupar os formulários envelopando com a tag <fieldset> (<legend> para dar uma legenda ao grupo / action=" " para onde os dados são enviados).
Segue um exemplo da anatomia: 


<form action="cadastro.php">                  !- action=" " para onde os dados são enviados -!
    <fieldset>                                !- para agrupar os formulários -!
        <legend>Dados Pessoais</legend>       !- legenda ao grupo -!
        <p>
            <label for="inome">Nome </label>  !- label for="inome" fazendo a ligação com o input do nome -!
            <input type="text" name="nome" id="inome" required placeholder="Digite seu nome"> !- input:text = pergunta o nome -!  
        </p>
        <p>
            <label for="isobrenome">Sobrenome </label> !- label for="isobrenome" fazendo a ligação com o input do sobrenome -! 
            <input type="text" name="sobrenome" id="isobrenome" required placeholder="Digite seu sobrenome"> 
        </p>                                                                   !- input:text para perguntar o sobrenome -!
        <p>
            <input type="submit" value="Enviar"> !- input:submit para criar um botão para enviar -!
            <input type="reset" value="Limpar">  !- input:reset para criar um botão para limpar -!
        </p>
    </fieldset>
</form>

Dados Pessoais

Existem vários tipos de input, agora vou citar alguns:

Tipos de input mais comuns

  1. input:hidden → uma entrada de formulário que está oculta, geralmente para armazenar valores que o usuário não precisa ver.
  2. input:text → define um campo de entrada de texto de linha única.
  3. input:password → define um campo de senha através de um asterisco.
  4. input:submit → define um botão para enviar dados do formulário.
  5. input:button → define um botão.
  6. input:radio → são conjuntos de botões que permite que o usuário selecione APENAS UMA das opções exibidas (colocar o mesmo name="" nas opções e id="" diferentes).
  7. input:checkbox → caixas de seleção permitem que um usuário selecione ZERO ou MAIS opções.
  8. input:search → é usado para campos de pesquisa.
  9. input:reset → define um botão de redefinição que redefinirá todos os valores do formulário para seus valores padrão.

Tipos de input com validação

  1. input:tel → é usado para inserir um número de telefone (pattern="^\(\d{2}\)\d{4, 5}-\d{4}$ → para aceitar somente números (XX)XXXX-XXXX ou (XX)XXXXX-XXXX).
  2. input:url → valida que a entrada é uma URL.
  3. input:email → valida que a entrada é um e-mail.
  4. input:number → define um campo de entrada numérica (você também pode definir restrições sobre quais números são aceitos).
  5. input:range → define um controle para inserir um número cujo valor exato não é importante (como um controle deslizante). O intervalo padrão é de 0 a 100 (no entanto, você pode definir restrições sobre quais números são aceitos com os atributos min, max, e :step).

Tipos de input de data

  1. input:date → qualquer entrada de data.
  2. input:month → permite selecionar um único mês.
  3. input:week → permite que o usuário selecione uma semana e ano.
  4. input:time → permite selecionar um horário específico.
  5. input:datetime-local → permite selecionar a combinação de data/hora.

Outros tipos de input

  1. input:file → No GET não funciona; define um botão para carregar um arquivo para o servidor (normalmente para enviar fotos).
  2. input:image → o mais estranho de todos, para botões gráficos de envio (o caminho para a imagem é especificado no atributo src=" ").
  3. input:color → é usado para campos de entrada que devem conter uma cor.

Existem 2 tipos de métodos de envio de formulário, GET (padrão) e o POST. Temos que adicionar o parâmetro method=" " para conseguimos mudar o método de envio.

  1. GET → é utilizado quando os dados não forem sensíveis (nome, peso, altura, etc...), quando utilizamos o método GET só podemos enviar até 3000 bits (aproximadamente 3000 letras).
  2. POST → é utilizado quando os dados são mais sensíveis (senha, foto, número de cartão de crédito, quando os dados excedem os 3000 bits, etc...). O mais ideal para a proteção de dados é utilizar o HTTPS (estudar mais pra frente HTTPS e RegEx → expressões regulares).

Output

A tag <output> é um elemento no qual um site ou aplicativo pode mostrar os resultados de um cálculo ou o resultado de uma ação do usuário, o objetivo do output é otimizar o produto e entragar mais valor para os usuários, é geralmente utilizado para representar o resultado de um cálculo. E para fazer uma ligação com o <input> utilizamos o parâmetro oninput=" " (serve para somente mostrar na tela, ele não é enviado junto com o formulário). 


<form action="cadastro.py">    
    <fieldset>
        <legend>Soma dos números</legend>
        <p>
            <label for="in1">Primeiro número: </label>                         !- label for="in1" ligação com input 1° número -!
            <input type="number" name="n1" id="in1" min="0" max="50" required  !- input:number p/ perguntar primeiro número -!
            oninput="isoma.innerHTML = Number(in1.value) + Number(in2.value)"> !- (JavaScript) pega o valor digitado -!
        </p>
        <p>
            <label for="in2">Segundo número: </label>                          !- label for="in2" ligação com input 2° número -!
            <input type="number" name="n2" id="in2" min="0" max="50" required  !- input:number p/ perguntar segundo número -!
            oninput="isoma.innerHTML = Number(in1.value) + Number(in2.value)"> !- (JavaScript) paga o valor digitado -!
        </p>
        <p>
            <label for="isoma">Soma: </label>                               !- label for="isoma" ligação com o output da soma -!
            <output name="soma" id="isoma">0</output>                       !- output p/ mostrar na tela a soma -!
        </p>
        <p>
            <input type="submit" value="Enviar">                            !- input:submit p/ criar botão para enviar -!
            <input type="reset" value="Limpar">                             !- input:reset p/ criar botão para limpar -!
        </p>
    </fieldset>
</form>
