CORES

Para colocar cores no site, utilizamos tags de estilo dentro da tag <style>. Conseguimos colocar as cores de algumas formas, pelo nome (em inglês), por código hexadecimais, RGB (Red, Green, Blue) ou HSL (Hue Saturation Lightness). Segue alguns exemplos de tags de estilo para cores:

  • color: → cor da letra
  • background-color: → cor do fundo
  • text-shadow: → sombra da letra
  • box-shadow: → sombra da caixa

Agora vou te mostrar como colocar as cores:

Por nome

Para colocar cores pelo nome, colocamos alguma tag de estilo de cor (color: / background-color:) dentro da tag <style>, escolhemos qual tag que vamos estilizar e digitamos a nome da cor em inglês.


        <body>                                               |    <style>
            <h1>Cor por nome</h1>                            |        h1 {
                <p>Exemplo de cores por nome</p>             |            background-color: blue;
        </body>                                              |        }

Por códido hexadecimal

Para colocar cores pelo código hexadecimal, colocamos alguma tag de estilo de cor (color: / background-color:) dentro da tag <style>, escolhemos qual tag que vamos estilizar, e digitamos o código da cor junto com um ( # ). Para ficar mais fácil, utilizamos o aplicativo GIMP (o mesmo para edição de fotos) para saber qual o código de cada cor. 


        <body>                                               |    <style>
            <h1>Cor por código hexadecimal</h1>              |        h1 {
                <p>Exemplo de cores por nome</p>             |            background-color: #0000ff;
        </body>                                              |        }

RGB e RGBA

Para colocar cores por RGB, colocamos alguma tag de estilo de cor (color: / background-color:) dentro da tag <style>, escolhemos qual tag que vamos estilizar e digitamos rgb(n°, n°, n°), primeiro n° é a cor vermelha (0 à 255), o segundo n° é a cor verde (0 à 255), e o terceiro é a cor azul (0 à 255). Para utilizar o RGBA, acrescentamos mais a letra a e mais um número (rgba(n°, n°, n°, n°)) que seria o da transparência que vai de 0 à 1, 0 totalmente transparênte, 1 totalmente visível. 


        <body>                                               |    <style>
            <h1>Cor por RGB e RGBA</h1>                      |        h1 {
                <p>Exemplo de cores por nome</p>             |            background-color: rgba(0, 0, 255, 0.5);
        </body>                                              |        }

HSL e HSLA

Para colocar cores por HSL, colocamos alguma tag de estilo de cor (color: / background-color:) dentro da tag <style>, escolhemos qual tag que vamos estilizar e digitamos hsl(n°, n°%, n°%), primeiro n° o tom (0 à 255), o segundo n° é a saturação (0% à 100%), e o terceiro é a luminosidade (0% à 100%). Para utilizar o HSLA, acrescentamos mais a letra a e mais um número (hsla(n°, n°%, n°%, n°)) que seria o da transparência que vai de 0 à 1, 0 totalmente transparênte, 1 totalmente visível. 


        <body>                                               |    <style>
            <h1>Cor por HSL e HSLA</h1>                      |        h1 {
                <p>Exemplo de cores por nome</p>             |            background-color: hsla(240, 100%, 50%, 0.5);
        </body>                                              |        }

DEGRADÊ

Conseguimos fazer um degradê de cores de 2 tipos (em linha, em circulo), para ambos utilizamos a tag de estilo background-image: dentro da tag <style>, (sempre utilizar * {height: 100%;} na configuração global das CSS) e background-attachment: fixed; para deixar o fundo fixo (se não quiser, utilizar background-attachment: scroll; para o fundo rolar junto com o conteúdo).

Degradê em linha

Para fazer um degradê em linha utilizamos as tags de estilo background-image: linear-gradient(colocar as cores); + background-attachment: fixed; para deixar o fundo fixo, ou background-attachment: scroll; para o fundo rolar junto com o conteúdo. Conseguimos mudar a direção em que vai o degradê, adicionar os parâmetros (to top / to right / to bottom / to left) entre os parênteses do linear-gradient(); ou também conseguimos utilizar ângulo para o direcionamento (ex: 45deg / 120deg). Exemplo: 


<style>                                                        |<style>
    * {                                                        |    * {
        heigth: 100%;                                          |        heigth: 100%;
    }                                                         ou    }
    body {                                                     |    body {
        background-image: linear-gradient(60deg, black, white);|        background-image: linear-gradient(to top, black, white);
        background-attachment: fixed;                          |        background-attachment: fixed;
    }                                                          |    }
</style>                                                       |</style>

Degradê em circulo

Para fazer um degradê em circulo utilizamos as tags de estilo background-image: radial-gradient(circle, colocar as cores); + background-attachment: fixed; para deixar o fundo fixo, ou background-attachment: scroll; para o fundo rolar junto com o conteúdo. Exemplo: 


                                                <style>
                                                    * { 
                                                        heigth: 100%;
                                                    } 
                                                    body {
                                                        background-image: radial-gradient(circle, black, gray, white);
                                                        background-attachment: fixed;
                                                    }
                                                </style>
